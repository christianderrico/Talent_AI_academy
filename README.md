# Repository Esercizi & Materiale del Corso

Questo repository raccoglie tutto il materiale prodotto durante il percorso di formazione: esercizi, esercitazioni, progetti, appunti e sperimentazioni sviluppate dai partecipanti del corso.

L’obiettivo è avere uno spazio condiviso dove:
- conservare il codice sviluppato durante le lezioni;
- confrontare approcci e soluzioni diverse;
- collaborare e imparare dagli altri;
- costruire nel tempo una piccola raccolta di esempi e progetti utili.

---

## Struttura del repository

Ogni partecipante può organizzare il proprio materiale in una cartella dedicata oppure per argomento/esercitazione, a seconda delle necessità del corso.

Esempio:

```text
/
├── esercitazione-1/
├── esercitazione-2/
├── progetto-finale/
├── appunti/
└── nome-corsista/
